# Plasmate Coverage Suite (Top 100 real-world, agent-relevant pages)
#
# Goals:
# - Measure how often Plasmate can fetch + compile a useful SOM on real sites
# - Produce a public scorecard (website/docs/coverage.json)
# - Drive the roadmap based on real failures
#
# Notes:
# - This is intentionally diverse, not just "top sites by traffic".
# - Many high-traffic sites are intentionally hostile to automation.
# - Failures are valuable signal.
